E-Commerce Assessment Project
Overview

This project demonstrates the implementation of an E-Commerce Product Management System using HTML, CSS, JavaScript (ES6), Bootstrap, and TypeScript.
It is divided into three main user stories, each focusing on different front-end and TypeScript concepts.

User Story 1 – Product Showcase Home Page

Tech Used: HTML5, CSS3, Bootstrap

Created a responsive homepage layout using Bootstrap grid system.

Displayed featured products with image, name, and price using product cards.

Included a functional newsletter subscription form with validation.

Used CSS transitions and hover effects for better interactivity.

User Story 2 – Product Listing with API Fetch and Filtering

Tech Used: JavaScript (ES6), Fetch API, JSON

Fetched product data dynamically from products.json.

Displayed products using DOM manipulation.

Added filter and sort functionality by category and price.

Implemented async/await with error handling using try…catch.

Displayed loading indicators during API calls.

User Story 3 – Product Management Module (TypeScript)

Tech Used: TypeScript

Defined an interface IProduct with fields like id, name, category, price, and stock.

Created a Product class implementing the interface.

Added a decorator to log changes in product price or stock.

Stored products in a Map and iterated through them using iterators.

Successfully compiled TypeScript code to JavaScript using tsc.
